# 项目亮点

## 游戏行业业务场景明确

项目聚焦 Steam 玩家反馈、版本复盘、商店页优化和竞品拆解，贴近游戏策划、运营和用户研究的真实工作流。

## 从评论到报告的完整 AI 工作流

流程覆盖信息收集、数据清洗、统计分析、关键词提取、业务标签归因、四象限判断、报告生成和导出，能展示 AI 工具如何提升工作效率。

## 四象限模型辅助优先级判断

“玩家情感 × 热度”模型把评论转化为核心卖点、优先修复问题、潜力亮点、长尾问题和分歧反馈，帮助团队把玩家声音转化为行动优先级。

## 支持竞品对比

可以输入多款 Steam 游戏，自动输出好评率、平均游玩时长、正负关键词、核心卖点和优先问题，适合竞品分析与产品机会点提炼。

## 支持无 API Key 本地运行

项目默认使用模板生成报告，无 OpenAI API Key 也能完整运行；配置 API Key 后可开启自然语言增强。

## 可导出报告和 CSV

支持 CSV、Markdown 和 ZIP 导出，便于沉淀分析结论、放入作品集或同步给团队。
