# Steam Game Insight Copilot PRD

## 1. 产品定位

Steam Game Insight Copilot 是面向游戏策划、运营、用户研究和竞品分析的玩家反馈分析工具。它把 Steam 公开评论转化为可视化图表、业务标签、四象限优先级和 Markdown 报告。

## 2. 目标用户

- 游戏策划：定位玩法、平衡、内容量和新手引导问题。
- 游戏运营：提炼宣发卖点、复盘版本更新、整理玩家反馈。
- 用户研究：快速浏览大量评论并形成结构化结论。
- 求职者：展示 AI 工具链、数据分析和游戏业务理解。

## 3. 核心场景

1. 单款游戏复盘：输入 AppID，获取评论并生成报告。
2. 竞品对比：输入多款游戏，比较好评率、关键词、标签和卖点。
3. 面试演示：使用 Demo 数据完整展示分析链路。
4. 资料沉淀：导出 CSV、Markdown 和 ZIP，用于作品集或复盘文档。

## 4. 功能需求

### 4.1 游戏搜索与解析

- 支持游戏名称搜索。
- 支持纯数字 AppID。
- 支持 Steam 商店 URL。
- 搜索失败时提示用户，不中断应用。

### 4.2 评论获取

- 使用 Steam appreviews 接口。
- 支持分页，直到达到上限或返回空评论。
- 支持语言、评论类型、购买类型、排序方式。
- 请求设置 timeout 和 sleep。
- 不保存 steamid。

### 4.3 文本处理

- 去除 HTML、换行、重复空格。
- 过滤少于 5 字符的短评论。
- 生成情感、日期、游玩小时、评论长度字段。

### 4.4 数据分析

- 输出评论总数、好评数、差评数、好评率、平均/中位数游玩时长。
- 支持中英文关键词提取。
- 支持高有用评论排序。

### 4.5 业务标签

标签包括玩法体验、内容量、性能优化、价格/性价比、平衡性、新手引导、联机/匹配、BUG/崩溃、更新/运营、美术/音乐。

### 4.6 四象限模型

基于标签热度和标签内好评占比生成：核心卖点、优先修复问题、潜力亮点、长尾问题、分歧反馈。

### 4.7 报告与导出

- 单款游戏报告。
- 竞品对比报告。
- CSV、Markdown、ZIP 导出。
- 无 API Key 使用模板；有 API Key 可润色。

## 5. 非功能需求

- Python 3.10+。
- Streamlit 页面中文化。
- 模块化代码，不把逻辑堆在 app.py。
- 文件路径自动创建。
- 核心函数具备 pytest 测试。

## 6. 风险与处理

- Steam 请求失败：返回友好提示。
- 搜索无结果：提示换关键词。
- 语言筛选无评论：提示调整条件。
- 标签为空：隐藏相关图表并说明原因。
- 无 API Key：正常输出模板报告。

## 7. 成功指标

- 用户能在 1 分钟内从输入 AppID 到获得报告。
- Demo 模式无需网络即可展示完整链路。
- 报告能直接用于版本复盘或面试讲解。
